# Cybersecurity Portfolio | Seerla Mahesh

A premium, interactive, and highly optimized cybersecurity portfolio featuring a cyberpunk hacker aesthetic, built to showcase expertise in secure web development, Security Operations Center (SOC) operations, and Vulnerability Assessment & Penetration Testing (VAPT).

---

## 🚀 Features

* **Terminal Boot Loader Screen**: Realistic Linux-style boot sequence (`./boot_sequence.sh`) initial load animation with automatic transition.
* **Interactive CLI Terminal**: Functional mock CLI interface allowing users to query certifications, skills, projects, and contact info, clear history, list files, and download the resume.
* **D3/SVG Skill Tree**: An interactive, collapsing tree representation of technical capabilities (languages, databases, tools, operating systems).
* **Side-by-Side Timeline**: Responsive, dual-column vertical experience and education timeline grid mapping key internships and degrees.
* **Animated Stats counters**: Intersection-observed counter statistics supporting floating decimals (e.g. current CGPA 8.1 / 10).
* **Cyberpunk Visuals**: Scanlines overlay, customized glowing bracket buttons, neon proximity secret effects, and matrix particle canvas background.
* **Fully Responsive**: Fluid, mobile-first design leveraging responsive grids and dynamic sidebar collapsing mechanisms.

---

## 🛠️ Technologies

* **Core Structure & Logic**: HTML5, Vanilla ES6+ JavaScript
* **Styling (CSS)**: Custom Vanilla CSS (Variables, keyframe animations, responsive grid layouts)
* **Visuals**: SVG, FontAwesome icons, Canvas API
* **Fonts**: Inter, Fira Code (Google Fonts)

## 💻 Installation

To run this project locally:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Mahesh09s/Mahesh-portfolio.git
   ```
2. **Open index.html**:
   Open `index.html` directly in any modern web browser, or use a local static server:
   ```bash
   # Using Node.js live-server
   npx live-server
   
   # Using Python static server (if available)
   python -m http.server 8000
   ```

## 🌐 Deployment

The site is configured for static web hosting. You can deploy it instantly on:

* **Vercel**: Push your code to GitHub and import the project into Vercel.
